### 使用 hexo 搭建

```bash
npm install -g hexo-cli
```

#### hexo 相关命令

```bash
// 初始化
hexo init {name}
// 编译
hexo generate
// 本地运行 http://localhost:4000
hexo serve
```

#### 部署

```bash
hexo deploy
```

### Hexo 使用

```bash
hexo new article
```
